## **ğŸ”¹ 2. Applying Functions with `apply()`, `map()`, and `lambda`**  
Often, we need to transform column values by applying a function.  

### **ğŸ”¹ 2.1: Using `apply()`**  
âœ”ï¸ Works on both **Series and DataFrames**  
âœ”ï¸ Can apply **custom or built-in functions**  
âœ”ï¸ Used for **row-wise or column-wise operations**  

#### âœ… **Example: Convert Age to Age Groups**  
```python
import pandas as pd

# Sample dataset
data = {'Name': ['Alice', 'Bob', 'Charlie', 'David'],
        'Age': [25, 17, 40, 30]}

df = pd.DataFrame(data)

# Define a function for categorizing age
def age_category(age):
    if age < 18:
        return "Teen"
    elif 18 <= age < 35:
        return "Young Adult"
    else:
        return "Adult"

# Apply the function to the Age column
df['Age Group'] = df['Age'].apply(age_category)
print(df)
```

#### **ğŸ“ Output:**
```
     Name  Age    Age Group
0   Alice   25  Young Adult
1     Bob   17        Teen
2  Charlie   40       Adult
3   David   30  Young Adult
```

### **ğŸ”¹ 2.2: Using `map()` (For Single Column Transformations)**  
âœ”ï¸ Works **only on Series**  
âœ”ï¸ Used for simple **value-to-value mapping**  

#### âœ… **Example: Mapping Gender Labels**

### **ğŸ”¹ 2.3: Using Lambda Functions (Anonymous Functions)**

## **ğŸ”¹ 3. Encoding Categorical Variables**  
Machine learning models require **numerical inputs**, so categorical data must be encoded.  

### **ğŸ”¹ 3.1: Label Encoding (Ordinal Encoding)**  
âœ”ï¸ Converts categorical values into **integer labels**  
âœ”ï¸ Suitable for **ordered categories**  

#### âœ… **Example: Encoding Education Levels**
```python
from sklearn.preprocessing import LabelEncoder

df['Education'] = ['High School', 'Bachelor', 'Master', 'PhD']

encoder = LabelEncoder()
df['Education Encoded'] = encoder.fit_transform(df['Education'])
print(df)
```

#### **ğŸ“ Output:**
```
  Education      Education Encoded
0  High School                 0
1     Bachelor                 1
2       Master                 2
3         PhD                 3
```

---

### **ğŸ”¹ 3.2: One-Hot Encoding (Dummy Variables)**  
âœ”ï¸ Converts **categorical data** into multiple binary columns  
âœ”ï¸ Suitable for **non-ordered categories**  

#### âœ… **Example: Encoding Cities using `pd.get_dummies()`**
```python

```

```
1    Bob   17           0             0           1
2 Charlie   40           1             0           0
3  David   30           0             0           1
```

## **ğŸ”¹ 4. Binning and Discretization**  
âœ”ï¸ Converts continuous numerical values into **categorical bins**  
âœ”ï¸ Helps in **reducing complexity**  

#### âœ… **Example: Binning Age into Groups**
