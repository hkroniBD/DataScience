ğŸ“Œ The `info()` function provides **basic details** about the dataset, including:

âœ”ï¸ **Data types** (int, float, object, etc.)  
âœ”ï¸ **Missing values**  

#### âœ… **Example: Using `info()` on a dataset**

## **ğŸ”¹ 3. Correlation Analysis**  
ğŸ“Œ **What is Correlation?**  
Correlation measures the **relationship between numerical variables**.  

âœ”ï¸ **Positive correlation**: One variable increases, the other increases  
âœ”ï¸ **Negative correlation**: One variable increases, the other decreases  
âœ”ï¸ **No correlation**: No relationship  

### **ğŸ”¹ 3.1: Calculating Correlation using `corr()`**
```python
df.corr()
```

#### **ğŸ“ Output:**
```
           Age    Salary
Age      1.000    0.998
Salary   0.998    1.000
```
âœ”ï¸ **High correlation** (close to **1 or -1**) indicates a strong relationship.  

---

### **ğŸ”¹ 3.2: Visualizing Correlation with a Heatmap**  
```python
import seaborn as sns
import matplotlib.pyplot as plt

# Compute correlation matrix
corr_matrix = df.corr()

# Plot heatmap
plt.figure(figsize=(5, 3))
sns.heatmap(corr_matrix, annot=True, cmap='coolwarm', fmt=".2f")
plt.title("Correlation Heatmap")
plt.show()
```
âœ”ï¸ **Identifies multicollinearity**  
âœ”ï¸ Helps in **feature selection**
